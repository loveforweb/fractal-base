# LFW Fractal Base Project
A base project using [Fractal.build](https://fractal.build/)

## Installing Fractal CLI tool

```sh
npm i -g @frctl/fractal
```

## Running the project
Install dependencies

```sh
npm i
```

Run the project
```sh
gulp
```

## Generating new components and pages
```sh
npm run generate
```

## More information
